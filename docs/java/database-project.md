# 数据库项目实战总结

## 使用过哪些数据库？

使用过的非关系型数据库有 Redis、MongoDB。关系型数据库有Oracle、MySQL、PgSQL。

## 如果数据量很大，你们是怎么做的？

面对数据量大的表，会通过分片的方式进行存储。比如使用每个月进行分片，或者半年一次分片。具体的分片规则通过业务量来觉得。如果是日志或者订单、商品库存、认证日志会进行每月的分片，来提高查询效率。

## 如果需要做数据同步，你的方案是什么？

如果是涉及数据迁移，按照不同情况来设计方案。
1. 如果是相同数据库，数据语句和结构都可以通用，可以在夜间通过数据库工具，通过dump来进行批量迁移。
2. 如果是不同数据库，可能会涉及到数据类型的映射，以及插入语句SQL的不同。通过编写脚本，先批量查询数据，然后进行数据类型的转换，最后通过批量插入的方式进行迁移。
